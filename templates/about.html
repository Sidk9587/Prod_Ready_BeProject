About MindCare AI

An intelligent platform for mental health awareness and early detection

🎯 Objective

MindCare AI provides an intelligent early screening platform capable of identifying emotional distress, stress levels, and potential mental health concerns through multimodal analysis — combining text, image, live video, and chat interaction.

⚙️ Technology Stack

Built with modern AI and web technologies:

Python / Flask TensorFlow / Keras scikit-learn OpenCV NLTK / NLP Ollama LLM Web Speech API

🔍 System Features

  • Text Analysis — NLP-based depression and stress detection from written input
  • Image Analysis — Deep learning facial emotion recognition (7 classes)
  • WhatsApp Analysis — Mental stability assessment from exported chat history
  • Live Webcam — Real-time emotion detection via webcam stream
  • AI Chatbot — Empathetic conversational support with voice assistant

⚠️ Disclaimer

This platform is intended for awareness and early support purposes only. It does not replace professional medical advice, diagnosis, or treatment. If you are experiencing a mental health crisis, please contact a qualified healthcare professional or emergency services immediately.

🧠 How the AI Works

The text model uses TF-IDF vectorization combined with a machine learning classifier trained on mental health datasets. The image model is a Convolutional Neural Network (CNN) trained on the FER-2013 dataset for 7-class facial emotion recognition. The chatbot integrates with locally-running Ollama LLM models for real-time empathetic conversation, with a Web Speech API voice interface for hands-free interaction.
